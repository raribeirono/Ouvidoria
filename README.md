# Sistema de Ouvidoria (SQL)

Este é um projeto de sistema de ouvidoria desenvolvido em Python com conexão a banco de dados SQL. O objetivo do sistema é permitir o registro, listagem e gerenciamento de reclamações, sugestões ou elogios.

## 🛠️ Tecnologias Utilizadas
* Python 3
* Banco de Dados SQL

## 📂 Arquivos do Projeto
* `OuvidoriaSQL.py`: Arquivo principal contendo os menus e a lógica de interação com o usuário.
* `operacoesbd.py`: Módulo responsável pelas funções de conexão e execução de comandos diretamente no banco de dados.

## 🚀 Como Rodar o Projeto
1. Certifique-se de ter o Python instalado.
2. Configure as credenciais do seu banco de dados SQL dentro do arquivo `operacoesbd.py`.
3. Execute o arquivo principal:
   ```bash
   python OuvidoriaSQL.py
